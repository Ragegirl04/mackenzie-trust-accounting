# Mackenzie Investments - Trust Account Reconciliation System

## Project Overview
An automated trust account reconciliation system simulating real-world 
accounting workflows at an asset management firm. Built to demonstrate 
skills in Excel automation, VBA, and Power Automate directly aligned 
with trust accounting operations.

## Tools & Technologies
- Microsoft Excel (SUMIFS, VLOOKUP, IF, Pivot Tables, Pivot Charts)
- VBA Macros (automated formatting and discrepancy flagging)
- Power Automate (scheduled daily email alerts)
- GitHub (version control and documentation)

## Features
- 100+ transaction records across RRSP, TFSA and FHSA accounts
- Automated reconciliation using SUMIFS and VLOOKUP formulas
- IF formula to flag accounts as Reconciled, Under or Over
- VBA Macro to format report and highlight discrepancies in red/green
- One click button to run reconciliation report
- Power Automate flow sends daily discrepancy alert email at 9AM
- Interactive dashboard with Pivot Tables and Charts

## Project Structure
- Transactions — 100 raw transaction records
- Reconciliation — expected vs actual balances with status
- Discrepancy — flagged accounts with action required
- Dashboard — visual summary with Pivot Charts

## How to Use
1. Open Trust_Accounting.xlsm
2. Go to Reconciliation sheet
3. Click Run Reconciliation button
4. Review red/green highlighted discrepancies
5. Power Automate sends daily alert email automatically at 9AM
